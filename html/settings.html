Backup & Restore

Backup and restore user data using a USB stick.
Make sure the USB stick has enough disk space, preferably with fat32 file system.
User configuration, banks and pedalboards are always saved and overwritten.
Note that current user data will be deleted during restore!



Reboot & Update

Reboot the MOD.
Note that all unsaved pedalboard changes will be lost.



Start the MOD's restore/update mode, where you can manually update the MOD via USB Mass Storage mode.
Note that all unsaved pedalboard changes will be lost.

Audio Server

Automatically copy the audio data from an unconnected output to another.
With this enabled, the 2 outputs will receive the same data if one of them is unconnected.
Note that changing this option requires a reboot for changes to take effect.

The Duo X S/PDIF outputs can be set to mirror the analog output or be treated independently.
Mirror mode is default, where S/PDIF will contain the same audio signal as the analog outputs.
Enable this option to have 2 extra outputs on your pedalboard.
Note that changing this option requires a reboot for changes to take effect.

MOD runs at 128 frames by default, enable this option to run JACK with 256 frames instead.
Useful if you need the extra time for more plugin processing.
Obviously this doubles the latency, so only enable it if really necessary.

Services

Note: Each additional service you enable consumes a small amount of CPU.
Only enable services that you absolutely need.


User Interface

Show a few more options on the Web Interface, useful for developers.

Show option to list 'beta' plugins in the Plugin Store.
Note that beta plugins can be unstable and cause crashes. If you share a pedalboard containing beta plugins, it will only be available to users that explicitly accept pedalboards with beta plugins.
Just remember: things can break!

Show option to access "MOD Labs" plugins in the Plugin Store.
Please note that MOD Labs plugins are community maintained, can be unstable and can cause crashes. If you share a pedalboard containing MOD Labs plugins, it will only be available to users that explicitly accept pedalboards using MOD Labs plugins. Just remember: things can break!

Workarounds

If the controller displays locks up or stops working correctly, we can try to detect this and automatically trigger a restart.
This is not completely safe though, false positives can happen and the controller screen might be restarted by mistake.
A reboot is required in order for this option to take place.

Supported Control Chain Devices

Cleanup

Delete selected user data from the device.
This will PERMANENTLY DELETE the selected USER DATA. This action is IRREVERSIBLE. The device won't work properly until new items are created or installed.









Reflash controller

Reinstall the firmware for the controller board.
This option is here for support reasons; If the power to your device was cut while installing the controller firmware, your device may end up in a state where the controller is not fully functional.

While the reflashing is happening, please DO NOT DISCONNECT THE POWER to the device.
There is a RISK OF BRICKING THE DEVICE if the power is cut.

Control Chain Device Update

×

Your Control Chain device firmware is ready to be updated.

To begin, follow these steps:

  • Disconnect the device from the Control Chain/Ethernet port
  • Hold down the first footswitch on the Control Chain device
  • While the first footswitch is down, connect the device to the MOD Duo via USB
  • Wait for the "update completed" message
  • Disconnect the device from the USB port
  • Connect it to the Control Chain/Ethernet port once again

Note: The update should only take a few seconds.
